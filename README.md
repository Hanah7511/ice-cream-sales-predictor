# Ice Cream Sales Predictor

This project predicts ice cream sales based on temperature using a Linear Regression machine learning model.

## Features
- Linear Regression model built from scratch
- Data visualization
- Web interface
- GitHub version control
- Netlify deployment

## Technologies Used
- Python
- NumPy
- Matplotlib
- HTML
- JavaScript
